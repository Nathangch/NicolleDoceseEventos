# ✨ Nicolle Doces e Eventos

Uma landing page moderna, dinâmica e responsiva desenvolvida para divulgar os serviços da **Nicolle Doces e Eventos**, uma empresa especializada em festas infantis. O projeto visa apresentar o portfólio de bolos, salgados e equipe, além de facilitar o contato direto de clientes através do WhatsApp e Instagram.

## 🚀 Tecnologias Utilizadas
- **HTML5**: Estrutura semântica e abas (tabs) para navegação ágil.
- **CSS3 (Vanilla)**: Design premium focado no público infantil, utilizando variáveis de cores em tons pastéis, *glassmorphism* (efeito de vidro), animações suaves e total responsividade.
- **JavaScript**: Lógica leve construída no lado do cliente para alternar o conteúdo das páginas sem a necessidade de recargas (Single Page Application feeling) e controle de menu mobile.

## 🎨 Funcionalidades
- **Navegação em Abas (Tabs)**: Transição suave entre as seções (Início, Bolos, Salgados, Avaliações, Contato) sem carregamentos repetitivos.
- **Galeria Interativa**: Catálogo visual dos bolos com transições em escala e visualização de preços via hover.
- **Design Responsivo**: Adaptado para uma experiência imersiva e clara tanto em monitores ultrawide quanto em smartphones.
- **Integração de Contato**: Formulário interativo e direcionamento automático para links de contato direto com a equipe.
